«ПЛАТФОРМА НОЕВ КОВЧЕГ» · РЕСПУБЛИКА АРМЕНИЯ

Финансирование национальных проектов через токенизацию залоговой недвижимости Цифровая инвестиционная платформа · Закон РА HO-159-N · Регламенты ЦБ РА 7/01–7/05

СУТЬ В ОДНОЙ ФРАЗЕ
Собственник пледжит недвижимость → платформа выпускает 2-х траншевый токен → инвестор покупает за деньги → деньги идут в Фонд РА → Минфин финансирует нац. проекты → построенные объекты возвращают капитал инвестору → собственник получает актив + доход + бонус.
ТРИ СТОРОНЫ — ВСЕ В ВЫИГРЫШЕ
🏠 Собственник Актив остаётся у него + 2.75% годовой купон + бесплатная страховка + 0% налога на имущество + 7–8% бонус в конце + статус «Соинвестор Армении». Итого ≈ 4.4% годовых сверх альтернатив без потери владения.
💰 Инвестор Старший транш: 5.5% купон + гарантия Минфина РА. Младший транш: до 13.5% годовых через рост NAV пула. Реальное обеспечение армянской недвижимостью. 0% налога на крипто-прирост для физ. нерезидентов.
🇦🇲 Государство РА До €1 млрд внебюджетного финансирования нац. проектов без увеличения внешнего долга. Ускорение национальных программ на 5–7 лет. Финансовый суверенитет.

ЧЕТЫРЕ ФАЗЫ
Фаза 0 (мес 0–6) Учреждение СП, IP-оценка, подготовка документации, политическое согласование
Фаза 1 (мес 6–24) Пилот €100M в действующем HO-159-N. Хард-гейт: Постановление Правительства о бюджетной гарантии.
Фаза 2 (мес 24–60) Пакет поправок (закон о ЦБ РА + HO-159-N + новый закон о Фонде). ЦБ РА как контрагент. Эмиссия драма под пул. €1 млрд+ годового оборота.
Фаза 3 (мес 60+) Тираж на движимое имущество, IP, акции. Региональный масштаб (Грузия, Казахстан, ЕАЭС, ЕС).

1. Регуляторная чистота: в действующем праве РА (HO-159-N + 4 регламента ЦБ).
2. Win-Win-Win: все три стороны в плюсе одновременно — без потерь.
3. Канал диаспоры: уникальный продукт под 7 млн армян по миру. По образцу Israel Bonds ($50B+ за 75 лет).
4. Реальное обеспечение: недвижимость + страховка + гарантия Минфина РА.
5. Финсуверенитет РА: деньги без МВФ, без внешнего долга, без условий.
6. Двухфазная стратегия: быстрый старт + амбициозное масштабирование.
